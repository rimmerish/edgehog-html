What it does

Small box. Direct controls. Weird results.

Built around immediacy: buttons, knobs, samples, synth voices, loops and performance accidents that are actually useful.

01

Memory samples

Fast onboard sample sets for drums, vocal textures, glitches and one-shot fragments.

02

SD playback

Stream longer files from SD, hold loops, freeze moments and turn samples into playable material.

03

Synth engines

Polyphonic digital synthesis with filter, envelopes and modulation and a voice-style formant synth engine, all under physical control.

04

USB MIDI

Send notes, receive notes, map controls and use EH-01 with a DAW or browser companion tool.

05

Local browser midi console

The EH-01 includes an offline Web MIDI console that turns a browser into a deeper control surface and sequencer. Over USB MIDI, it provides live sliders, note triggering, MIDI clock, preset storage, pattern saving, scene recall, song chaining, scale controls, and an assignable XY pad.

06

Make it yours

EH-01 is built to be customised. Swap knobs, change keycaps, experiment with colours, and use the included 3D-printable part files to make the instrument feel like your own.

Computer UI.

The EH-01 includes an offline Web MIDI console that turns a browser into a deeper control surface and sequencer.

The sequencer supports 32 steps, velocity, probability, pattern variations, smart mutate, motion sequencing, and per-step CC locks. This allows notes and parameter changes to be sequenced together, giving the EH-01 browser-assisted parameter-lock behaviour in a compact, local, no-install interface.

About

About me.

Designed, built and tested by hand.

Hi, my name is Leo Rolph. I’m a designer working from a small home studio in Fremantle, Western Australia. With a strong background in graphic and product design, I’ve always had a small obsession with electronics, synths and sound generation.

EH-01 is the result of multiple years of experimentation, many synth prototypes and a long string of design choices. It did not start as a commercial product. It started as an instrument I wanted for myself: small, capable, non-standard, immediate and built around several different sound systems I could use for my own sound generation.

The final EH-01 is an attempt to give other people the same kind of hands-on joy I get from playing with it.
